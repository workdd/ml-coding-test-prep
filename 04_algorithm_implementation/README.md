# 04_algorithm_implementation

## 응용 ML 케이스 문제

이 폴더에는 실제 데이터와 문제 상황에 적용할 수 있는 머신러닝 응용 문제들이 포함되어 있습니다.

### 예시 문제

-   추천 시스템 구현
-   시계열 예측
-   텍스트 분류
-   이미지 분류 파이프라인

### 파일 구성

-   `problem.py`: 문제 설명 및 함수 템플릿
-   `solution.py`: 예시 해답 코드
-   `test.py`: 테스트 케이스

### 사용 방법

1. `problem.py`의 설명을 읽고 직접 응용 문제를 해결해보세요.
2. `test.py`로 구현한 코드의 동작을 검증할 수 있습니다.
