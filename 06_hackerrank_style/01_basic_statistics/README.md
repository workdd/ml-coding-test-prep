# 📊 기본 통계 문제들

HackerRank에서 자주 출제되는 기본 통계 문제들입니다.

## 문제 목록

### 1. 기술통계량 계산 🟢
- **파일**: `descriptive_statistics.py`
- **주제**: 평균, 중앙값, 최빈값, 표준편차
- **난이도**: Easy

### 2. 정규분포 확률 계산 🟡
- **파일**: `normal_distribution.py`
- **주제**: 정규분포, Z-score, 확률 계산
- **난이도**: Medium

### 3. 이항분포 문제 🟡
- **파일**: `binomial_distribution.py`
- **주제**: 이항분포, 조합, 확률질량함수
- **난이도**: Medium

### 4. 상관계수 계산 🟢
- **파일**: `correlation_coefficient.py`
- **주제**: 피어슨 상관계수, 공분산
- **난이도**: Easy
